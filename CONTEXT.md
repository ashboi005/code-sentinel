# CodeSentinel Glossary

## CodeSentinel

An active web defense agent that helps developers understand, test, and fix security-impacting coding mistakes before production.

## Proxy Backend

A stateless model gateway that forwards authorized CodeSentinel model requests to Groq without storing chat history, repository data, scan state, or reports.

## Agentic CLI

The local command-line application that runs the CodeSentinel agent, invokes local tools, coordinates OpenHarness, and writes reports.

## Scan Target

The local path or deployed URL that CodeSentinel is asked to inspect. Phase 1 supports local path targets only.

## Report

The Markdown artifact generated by CodeSentinel to summarize observed evidence, current limitations, and recommended next steps.
